---
description: interactive ai assistant workflow
---
1. Add any Creative Render node.
2. Click the "Ask AI Assistant" button.
3. Type a prompt like: "Create a plasma effect with neon purple and cyan colors."
4. Click "OK".
5. The AI will generate the code and inject it into the editor.
6. Sliders for any detected uniforms will be created automatically.
7. If there are errors, click "Ask AI Assistant" again and say "Fix the errors in the current code".
